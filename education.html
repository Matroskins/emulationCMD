Компания «СоларЛаб» обучает и трудоустраивает начинающих разработчиков и тестировщиков. Нам не важен ваш пол, возраст или образование, только знания и навыки.

.net

Требования:

  • базовые знания языка С#
  • понимание основ ООП
  • базовое понимание реляционных баз данных
  • будет плюсом: html, css,js, опыт с linq или веб-сервисами

На курсе вы научитесь:

  • разработке web-ориентированных решений с многоуровневой архитектурой на платформе .NET Framework
  • применению современных паттернов проектирования.
  • нюансам создания качественного ПО с помощью unit-тестов и непрерывной интеграции (CI)
  • работе по методологии SCRUM
  • ведущие разработчики компании поделятся опытом использования технологий ASP.NET MVC, Web API, WCF, Entity Framework

По окончанию курса компания "СоларЛаб" предлагает работу лучшим выпускникам.

Если хотите записаться на курс, пришлите краткое письмо с описанием ваших знаний и опыта на адрес education@solarl.ru.
