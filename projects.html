Компания СоларЛаб является участником партнерской программы компании Microsoft, и большинство наших проектов реализовано на стеке технологий компании Microsoft:

  • платформе .NET
  • СУБД Microsoft SQL Server
  • фреймворках ASP.NET MVC
  • Entity Framework
  • Windows Communication Foundation
  • Web API

Так как большинство разрабатываемого нами ПО – это веб-приложения, то также нам приходится плотно работать с фронтендом - версткой HTML, CSS, Javascript.

Также мы занимаемся мобильной разработкой под платформу iOS.

В области тестирования программного обеспечения мы стремимся к автоматизации рутинных процессов, и развиваем автоматизацию нагрузочного и функционального тестирования с использованием фреймворков Selenide и Cucumber.

Большинство наших проектов разрабатывается по Scrum методологии, с двухнедельными спринтами. Мы используем следующие атрибуты Scrum процесса: планирование, ретроспектива, покер, измерение скорости команды.
